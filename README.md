## 📂 Repository Structure

* `payloads.txt`: The main wordlist containing various SQL injection payloads categorized by type (Auth bypass, Union based, Time-based blind, Error-based, etc.).
* `README.md`: Project documentation and guidelines.
* `LICENSE`: MIT License.

## 🚀 How to Use

These payloads can be used with security testing tools such as:
* **OWASP ZAP** or **Burp Suite** (Intruder/Fuzzing)
* **SQLMap** (Custom wordlists)
* Custom automated security scanners/WAF validation scripts
